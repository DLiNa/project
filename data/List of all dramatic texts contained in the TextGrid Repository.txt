# (uploaded on June 17, 2015)

# Info on how this list was compiled can be found in our blog post.
# As described in the post, we decided to bundle 5 dramatic pieces that
# consist of several parts and glued them together in a new XML file:
# - Arnim: Halle und Jerusalem,
# - Goethe: Faust, Teil 1,
# - Hebbel: Nibelungen,
# - Immermann: Alexis,
# - Schiller: Wallenstein.
# Plus, we had to delete the two original pieces contained in the Repository
# (a French and an Italian one). So now our list only contains the
# German-language texts of the genre 'drama'.
# We uploaded the 666 XML files to our Github here:
# https://github.com/DLiNa/project/tree/master/data/textgrid-repository-dramas
